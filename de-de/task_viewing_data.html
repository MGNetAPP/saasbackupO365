    Anzeigen von Daten mit NetApp SaaS Backup für Office 365 Bearbeiten Sie auf GitHub Dokumentänderungen anfordern

    19.04.2013 Mitwirkende

    Sie können Daten nach bestimmten Ergebnissen filtern, die Inline-Suche verwenden, um das zu finden, was Sie benötigen, und das Aktivitätsprotokoll mit einer Liste aller Aktivitäten anzeigen.

    Erstellen eines benutzerdefinierten Filters

    Sie können die Ansicht Ihrer Postfächer, Websites oder MySites filtern, um nur Ergebnisse anzuzeigen, die bestimmten Kriterien entsprechen. Beispielsweise können Sie Ihre Filter so einstellen, dass nur Postfächer in einem bestimmten Land und einer bestimmten Abteilung in diesem Land angezeigt werden.

    Schritte

    1. Klicken Sie im Dashboard in dem Feld des Dienstes, für den Sie einen Filter erstellen möchten, auf die Nummer über PROTECTED oder UNPROTECTED .
      Die Zahl über PROTECTED gibt die Anzahl der Postfächer, MySites oder Gruppen an, die derzeit durch eine Sicherungsrichtlinie geschützt sind. Die Zahl über UNPROTECTED gibt die Anzahl der Postfächer, MySites oder Gruppen an, die nicht durch eine Sicherungsrichtlinie geschützt sind.

    2. Klicken Sie auf Filter .

    3. Klicken Sie auf das Select Dropdown-Menü, und wählen Sie die Filter Ihrer Wahl.
      Für Microsoft SharePoint Online können Sie nach Vorlagen-ID filtern. Sie können die Vorlagen-ID eingeben, um danach zu suchen, oder sie aus dem Dropdown-Menü auswählen.
      Für alle anderen Dienste können Sie nach Gruppe, Land, Büro, Abteilung, Titel, Domain oder Land filtern. Wenn Sie Sicherheitsgruppen haben, werden diese auch als Filteroptionen aufgeführt. 

       Das zweite Dropdown-Menü enthält eine Auswahl, die auf dem ausgewählten Filter basiert. Wenn Sie beispielsweise Gruppe als ersten Filter auswählen, können Sie eine der Ebenen der Sicherungsrichtliniengruppen als sekundären Filter auswählen.
  image: select_filter.jpg [Screenshot der ersten Filteroptionen]
  Ein Suchfeld wird angezeigt.
. Geben Sie Ihre Suchkriterien ein.
. Wenn Sie weitere Filter hinzufügen möchten, klicken Sie auf * Filter hinzufügen * und treffen Sie Ihre Auswahl.
. Klicken Sie auf * Filter anwenden *.
  Filterergebnisse werden angezeigt.

    Durchführen einer Suche in Microsoft Exchange Online

    Sie können eine Inline-Suche in einem einzelnen Postfach nach bestimmten Inhalten durchführen. Dies gilt auch für Postfächer, die Teil einer Office 365-Gruppe sind.

    Schritte

    1. Aus dem Armaturenbrett, klicken Sie auf die Zahl über geschützte oder ungeschützte in Exchange - Box.

    2. Wenn Sie nach GESCHÜTZTEN Postfächern suchen, klicken Sie auf die E-Mail-Adresse, nach der Sie suchen möchten .

      1. Wählen Sie die Kategorie (E-Mail, Aufgaben, Kontakte, Kalender oder Andere) aus, die Sie durchsuchen möchten.

      2. Wählen Sie eine Datei oder einen Ordner aus Dropdown - Menü neben dem Suchfeld , um anzuzeigen , wenn Sie für eine Datei oder einen Ordner suchen.

      3. Geben Sie eine Suchzeichenfolge in das Suchfeld ein.
        Die Suche wird automatisch durchgeführt und die Ergebnisse werden angezeigt, nachdem der Suchbegriff eingegeben wurde.

    3. Wenn Sie nach UNGESCHÜTZTEN Postfächern suchen, wählen Sie das Postfach aus, das Sie durchsuchen möchten.

      1. Geben Sie eine Suchzeichenfolge in das Suchfeld ein.
        Die Suche wird automatisch durchgeführt und die Ergebnisse werden angezeigt, nachdem der Suchbegriff eingegeben wurde.

    Durchführen einer Suche in Microsoft OneDrive for Business

    Sie können eine Inline-Suche innerhalb einer einzelnen MySite nach bestimmten Inhalten durchführen.

    Schritte

    1. Klicken Sie im Dashboard auf die Nummer über dem Feld PROTECTED in OneDrive.

    2. Klicken Sie auf die MySite, in der Sie die Suche durchführen möchten.

    3. Klicken Sie auf die Dateien, die Sie durchsuchen möchten.

    4. Geben Sie eine Suchzeichenfolge in das Suchfeld ein.
      Die Suche wird automatisch durchgeführt und die Ergebnisse werden angezeigt, nachdem der Suchbegriff eingegeben wurde.

    Durchführen einer Suche in Microsoft SharePoint Online

    Sie können eine Inline-Suche innerhalb einer einzelnen MySite nach bestimmten Inhalten durchführen. Dies gilt auch für Sites, die Teil einer Office 365-Gruppe sind.

    Schritte

    1. Klicken Sie im Dashboard auf die Nummer über dem Feld PROTECTED in SharePoint.

    2. Klicken Sie auf die Site, auf der Sie die Suche durchführen möchten.

    3. Klicken Sie auf die Inhaltskategorie, die Sie durchsuchen möchten.

    4. Geben Sie den Suchbegriff in das Suchfeld ein.
      Die Suche wird automatisch durchgeführt und die Ergebnisse werden angezeigt, nachdem der Suchbegriff eingegeben wurde.

    Auftragsverlauf anzeigen

    NetApp SaaS Backup für Office 365 speichert ein Protokoll aller Aufträge, das den Auftragstyp, den Dienst, die Startzeit, die Endzeit und den Abschlussstatus enthält.

    Schritte

    1. Klicken Sie im linken Navigationsbereich auf Berichterstellung .

      Eine Liste aller SaaS Backup - Jobs unter dem Auftragsverlauf Registerkarte angezeigt.

    2. Klicken Sie auf Filter, um die Ergebnisse zu filtern.

    3. Klicken Sie auf das Select Dropdown-Menü, und einen Filter auswählen.
      Sie können nach Richtlinie, Dienst oder Typ filtern. Nachdem Sie einen Filter ausgewählt haben, wird ein Suchfeld angezeigt.

    4. Geben Sie Ihre Suchkriterien ein.

    5. Wenn Sie weitere Filter hinzufügen möchten, klicken Sie auf Filter hinzufügen.

    6. Klicken Sie auf Filter anwenden .

      Filterergebnisse werden angezeigt.

    7. Klicken Sie auf einen Job, um die Ansicht für zusätzliche Jobdetails zu erweitern.

    Anzeigen des Aktivitätsprotokolls

    Ein Protokoll wird über alle Aktivitäten in SaaS Backup für Office 365 gespeichert. Das Protokoll enthält das Datum jeder ausgeführten Aktion sowie den Namen des Benutzers, der die Aktion ausgeführt hat. Sie können das Aktivitätsprotokoll nach Service und Ereignis filtern. Wenn Sie beispielsweise alle Wiederherstellungsvorgänge für Microsoft Exchange Online anzeigen möchten, können Sie das Aktivitätsprotokoll filtern, um diese spezifischen Ergebnisse anzuzeigen.

    Schritte

    1. Klicken Sie im linken Navigationsbereich auf Berichterstellung .

    2. Klicken Sie auf die Registerkarte Aktivitätsprotokoll .
      Eine Liste aller Aktivitäten von SaaS Backup für Office 365 wird angezeigt.

    3. Klicken Sie auf Filter, um die Ergebnisse zu filtern.

    4. Klicken Sie auf das Select Dropdown-Menü, und einen Filter auswählen.
      Sie können nach Service oder Ereignis filtern. Nachdem Sie einen Filter ausgewählt haben, wird ein Suchfeld angezeigt.

    5. Geben Sie Ihre Suchkriterien ein.

    6. Wenn Sie weitere Filter hinzufügen möchten, klicken Sie auf Filter hinzufügen.

    7. Klicken Sie auf Filter anwenden .

      Filterergebnisse werden angezeigt.

    Herunterladen des Aktivitätsprotokolls

    Ein Protokoll wird über alle Aktivitäten in SaaS Backup für Office 365 gespeichert. Das Protokoll enthält das Datum jeder ausgeführten Aktion sowie den Namen des Benutzers, der die Aktion ausgeführt hat. Sie können das Aktivitätsprotokoll in eine CSV-Datei herunterladen.

    Schritte

    1. Klicken Sie im linken Navigationsbereich auf Berichterstellung .

    2. Klicken Sie auf die Registerkarte Aktivitätsprotokoll .
      Eine Liste aller Aktivitäten von SaaS Backup für Office 365 wird angezeigt.

    3. Klicken Sie auf Download.

      Das Aktivitätsprotokoll wird als CSV-Datei heruntergeladen.

    Herunterladen eines Protokolls der abgeschlossenen Aufträge

    Sie können eine Excel-Tabelle mit erfolgreich abgeschlossenen Jobs herunterladen.

    Schritte

    1. Klicken Sie im linken Navigationsbereich auf Jobs

    2. Klicken Sie auf den kürzlich abgeschlossenen Job, den Sie herunterladen möchten.

    3. Klicken Sie unter der Anzahl der erfolgreich abgeschlossenen Aufträge auf Erfolgreich .

    4. Klicken Sie oben rechts auf Download .
      Das Protokoll wird heruntergeladen
