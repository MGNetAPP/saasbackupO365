    Sicherheitsgruppen hinzufügen Bearbeiten Sie auf GitHub Dokumentänderungen anfordern

    08/06/2019 Mitwirkende

    Sicherheitsgruppen können als Filteroptionen zum Anzeigen Ihrer Daten und zum Erstellen von Regeln verwendet werden.

    Sie können bis zu 3 Sicherheitsgruppen hinzufügen. Sie können dann Ihre Sicherheitsgruppen als Filteroptionen in SaaS Backup verwenden.

    Neue Sicherheitsgruppen müssen durch eine automatische Synchronisierung oder eine manuelle Synchronisierung erkannt werden, bevor sie hinzugefügt werden können.
    Erstellen, bearbeiten oder löschen Sie eine Sicherheitsgruppe im Admin Center .

    Schritte

    1. Klicken Sie auf Kontoeinstellungen .

    2. Klicken Sie auf SICHERHEITSGRUPPEN .

    3. Geben Sie im Suchfeld den Namen der Sicherheitsgruppe ein, die Sie hinzufügen möchten.

    4. Klicken Sie auf Hinzufügen .
